# Changelog

All notable changes to this extension will be documented in this file.

## [1.0.0] - 2024-11-07

### Added
- Initial release of Cloudinary Background Removal extension
- Remove backgrounds from images using Cloudinary's AI-powered processing
- Support for multiple background removal methods with automatic fallback
- Auto file detection from Finder
- Real-time progress updates during processing
- Support for PNG, JPG, JPEG, GIF, and WEBP image formats
- User-configurable preferences for Cloudinary cloud name, upload preset, and output directory
